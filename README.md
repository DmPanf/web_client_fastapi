# web_client_fastapi
Web-Client for FastAPI Server with Image Processing


### Структура файлов для простого Web-приложения

- **Чтобы сделать сайт доступным на другом порту (ex. 8080), нужно изменить конфигурацию веб-сервера (nginx.conf)**

<code>
my-simple-website/
│
├── index.html
│
├── css/
│   └── styles.css
│
├── js/
│   └── scripts.js
│
├── Dockerfile
│
├── docker-compose.yml
│
└── nginx.conf
</code>
